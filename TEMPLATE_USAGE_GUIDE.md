# Email Template System - User Guide & Examples

## Quick Start (5 Minutes)

### Step 1: Navigate to Compose
Click "Compose Email" from the dashboard navigation.

### Step 2: Choose Your Mode
The page defaults to "Use Professional Template" (recommended).
- **Professional Template:** Pre-written, category-specific emails
- **Custom Email:** Write your own subject and body

### Step 3: Select Your Category
Choose from 8 professional templates:

```
[Developer (GitHub)      ] - Tech professionals
[Web (General)          ] - General business
[Gaming                 ] - Gaming community
[Marketing              ] - Marketing professionals
[Business               ] - Executives & founders
[Freelancer             ] - Freelancers
[Blogger                ] - Content creators
[Job Seeker             ] - Job seekers
```

### Step 4: Enter Your Link
Paste your website or offer link. This becomes the main call-to-action.

Example links:
- `https://yoursite.com/offer`
- `https://yoursite.com/webinar`
- `https://yoursite.com/affiliate?id=12345`
- `https://example.com/apply`

### Step 5: Customize (Optional)
Expand "Customize Sender Details" to add:
- Your name
- Your email address
- Your phone number

### Step 6: Preview
Click "Preview Template" to see how the email will look.

### Step 7: Send
Click "Send to Pending Leads"

## Personalization Features

### Automatic Personalization
The system automatically inserts:
- **Recipient Name** - From the database (e.g., "John Smith" replaces {{NAME}})
- **Your Link** - From the form (e.g., your website URL)
- **Your Contact Info** - Name, email, phone

### Result
Each recipient receives a personalized email that feels direct and personal, not mass-sent.

**Example:**
- Database recipient: john_smith@example.com, name: "John Smith"
- Template placeholder: "Hi {{NAME}},"
- Sent email: "Hi John Smith,"

## Best Practices

### 1. Match Template to Audience
- **Developer (GitHub):** Use for GitHub users, engineers, technical folk
- **Web (General):** Use for mixed business audience
- **Gaming:** Use only for gaming/streaming audience
- **Marketing:** Use for marketing professionals
- **Business:** Use for executives/founders
- **Freelancer:** Use for freelancers/contractors
- **Blogger:** Use for content creators/influencers
- **Job Seeker:** Use for job-seeking audience

### 2. Compelling Link Strategy
- Link to something valuable (webinar, resource, offer)
- Use UTM parameters for tracking: `?utm_campaign=marketing`
- Include clear value proposition in URL

Examples:
```
https://mysite.com/free-webinar
https://mysite.com/download-guide?utm_campaign=devs
https://mysite.com/pricing?offer=50off
https://mysite.com/job-posting
```

### 3. Sender Details Matter
- **Name** - Real name builds trust
- **Email** - Must be real/monitored email
- **Phone** - Increases response (optional but recommended)
- **Title** - Shows credibility (especially for Business)

### 4. Always Preview
- Check subject line
- Verify link is correct
- Ensure all placeholders are filled
- Review for typos

### 5. Send in Batches
- Start with smaller test sends
- Monitor response rates
- Refine link/offer based on results
- Scale successful campaigns

## Troubleshooting

### "Link field is required"
**Problem:** You selected Template mode but didn't enter a link.
**Solution:** Paste your website URL in the "Your Website Link" field.

### "No pending leads to send to"
**Problem:** All leads in that category have already been emailed.
**Solution:** 
- Check dashboard for pending status
- Select "All Categories" to see all pending leads
- Reset lead status in database if needed

### "Emails sent: 0"
**Problem:** Template sent but no emails delivered.
**Solution:**
- Check SMTP settings
- Verify email addresses in database
- Check spam folder
- Review application logs

### Preview not loading
**Problem:** "Preview Template" button doesn't show content.
**Solution:**
- Wait a moment for preview to load
- Refresh the page
- Try selecting category again
- Check browser console for errors

## Success Metrics

After sending campaigns, track:

1. **Open Rate** - % who opened email
2. **Click Rate** - % who clicked your link
3. **Response Rate** - % who replied
4. **Conversion Rate** - % who completed desired action

Typical benchmarks:
- Open Rate: 20-40%
- Click Rate: 2-5%
- Response Rate: 1-10%
- Conversion Rate: 0.5-2%
